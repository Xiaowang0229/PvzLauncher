# 常见问题解答 Q&A

#### Q1.在下载游戏时报错 `The SSL connection could not be established, see inner exception.`

此问题通常由系统代理引起，请尝试关闭代理后再次尝试下载游戏

#### Q2. 在下载游戏时报错 `由于目标计算机积极拒绝，无法连接。（lz.quia.top:443）`

此问题由启动器使用的**直链解析服务**异常引起的，请耐心等待服务恢复正常

#### Q3.更新通道改成Development后如何改回去

一般来说，由于避免某些bug的产生，我们将测试版的通道锁定在了development

#### Q4.要是能和改版深度合作，然后每个版本都拆碎成一个一个包，玩家可以自由选择往里加，就像大多数游戏打mod那样就好了 Q5.我想提一个建议，就是关于融合版它有模组，能不能直接在融合版的基础上增加下载模组的功能

目前近期暂无计划针对某一特定版本做改动

#### Q6.可以一次性开多个修改器/游戏吗？

不可以，我们的程序和pvz原版一样，只允许单开游戏或者启动器

#### Q7.为什么点击启动修改器后，修改器并未正确显示？

有些修改器必须先开游戏本体才能正常启动，检查你是否运行了游戏

#### Q8.You must install or Update.net to run this application

安装评论区提供的运行时即可

#### Q9.支持32位系统吗

支持

#### Q10.The process cannot access the file 'AppData\Local\Temp\PVZLAUNCHER.UPDATE.CACHE' because it is being used by another process.

更新包被杀毒软件占用，与此程序无关
